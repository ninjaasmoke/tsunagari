### SETUP

1. install GO

### RUN

1. `go run main.go`

### TEST

Pass the following payload to the `/createRequest` endpoint

Example:
```{
    "external_url": "https://somenew.free.beeceptor.com/fetch",
    "rest_method": "POST",
    "request_body": "{\"customer_params\":[{\"name\":\"Loan Number\",\"value\":\"5567873\"},{\"name\":\"Customer Id\",\"value\":\"CUST8472\"}]}",
    "request_headers": "{\"Content-Type\":\"application/json\"}",
    "request_ttl": 150,
    "response_call_back": "https://somenew.free.beeceptor.com/callback",
    "ref_id": "INT1234567890"
}
```
| Fields                |  Explained
|---|---|
| external_url          |  biller API
| rest_method           |  GET, POST, etc
| request_body          |  any request body to sent to biller
| request_headers       |  any headers to be sent to biller
| requerst_ttl          |  timeout the reverse proxy must wait till it gets a response from biller API
| response_call_back    |  the URL that the reverse proxy must call with whatever response it gets from biller API
| ref_id                |  the identifier to uniquely identify a request

### RESPONSE

Example:
```
{
    "status": "success",
    "status_code": 201,
    "message": "Request Created",
    "ack_id": "TSUNA3953ccb3-ffb6-4443-8998-d036c4aa6129",
    "request_data": {
        "external_url": "https://somenew.free.beeceptor.com/fetch",
        "request_body": "{\"customer_params\":[{\"name\":\"Loan Number\",\"value\":\"5567873\"},{\"name\":\"Customer Id\",\"value\":\"CUST8472\"}]}",
        "rest_method": "POST",
        "request_headers": "{\"Content-Type\":\"application/json\"}",
        "response_call_back": "https://somenew.free.beeceptor.com/callback",
        "request_ttl": 150,
        "ref_id": "INT1234567890"
    }
}
```
| Fields        |   Explained
|---|---|
| status        |   The status of the request in the reverse proxy (will be successful if it was able to call the API)
| status_code   |   The status code of the request in the reverse proxy
| ack_id        |   A unique ID generated by the reverse proxy
| request_data  |   The data that the reverse proxy uses to send the request to biller
